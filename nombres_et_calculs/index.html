    Question 1¶

    Résolution de l'équation du premier degré $ax + b = cx + d$

    Pour résoudre cette équation, suivez ces étapes :

    1. Regroupez les termes similaires :

      • Soustrayez $x$ des deux côtés de l'équation. $ ax + b - cx = cx + d - cx $

    2. Simplifiez l'équation :

      • Factorisez le côté gauche si possible. $x(a - c) + b = d$

    3. Isoler la variable $x$ :

      • Soustrayez $b$ des deux côtés de l'équation. $x(a - c) + b - b = d - b$

    4. Divisez par le coefficient de $x$ :

      • Divisez chaque côté par $(a - c)$ pour isoler $x$.
        $x = \frac{{d - b}}{{a - c}}$

    C'est la solution de l'équation du premier degré $ax + b = cx + d$ avec la condition $a \neq c$
